دليل الحصول على مفتاح API

اتبع هذه الخطوات البسيطة لتفعيل المولّد في أقل من دقيقة.

  1. 1

    الانتقال إلى Google AI Studio

    انقر على الرابط التالي للذهاب مباشرةً إلى صفحة إنشاء مفاتيح API.

    اذهب إلى صفحة مفاتيح API
  2. 2

    إنشاء مفتاح جديد

    في الصفحة التي ستفتح، انقر على زر "Create API key ".

  3. 3

    نسخ المفتاح

    فورًا، سيظهر لك مفتاحك السري. انقر على أيقونة النسخ بجانبه.

  4. 4

    لصق المفتاح والعودة

    ارجع إلى صفحة المولّد، والصق المفتاح في الحقل المخصص له.

مهم جدًا

مفتاح الـ API الخاص بك سري جدًا. لا تشاركه مع أي شخص أبدًا.

مشاكل وحلول شائعة

أسئلة قد تواجهك أثناء استخدام الأداة.

تأكد من نسخ المفتاح بالكامل بدون أي مسافات زائدة في البداية أو النهاية. إذا استمرت المشكلة، قد يكون هناك ضغط على خوادم Google، حاول مجددًا بعد فترة قصيرة.

نعم، إنه آمن تمامًا. يتم حفظ مفتاحك في متصفحك فقط ولا يتم إرساله أو تخزينه في أي مكان آخر. عند إغلاق المتصفح، يتم حذفه تلقائيًا.

نعم، يوجد باقة مجانية بسعة استخدام كبيرة جدًا ومناسبة للاستخدام الشخصي والمطورين. لن تحتاج إلى الدفع إلا في حالة الاستخدام التجاري الضخم.
