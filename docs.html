دليل الحصول على مفتاح API

اتبع هذه الخطوات البسيطة لتفعيل المولّد في أقل من دقيقة.

  1. 1

    الانتقال إلى Google AI Studio

    انقر على الرابط التالي للذهاب مباشرةً إلى صفحة إنشاء مفاتيح API.

    اذهب إلى صفحة مفاتيح API
  2. 2

    إنشاء مفتاح جديد

    في الصفحة التي ستفتح، انقر على زر "Create API key".

  3. 3

    نسخ المفتاح

    فورًا، سيظهر لك مفتاحك السري. انقر على أيقونة النسخ بجانبه.

  4. 4

    لصق المفتاح والعودة

    ارجع إلى صفحة المولّد، والصق المفتاح في الحقل المخصص له.
